Instructions-

1. Unzip the folder PartA_KnowledgeGraphApp.zip
2. To run the application, navigate into knowledge_graph folder and run the bash script './run.sh' and follow the instructions given in PartA_TaskA_KnowledgeGraphApp_Instruction.pdf under Quick Start section.
3. The data folder inside knowledge_graph contains a sample csv file to test the upload feature.
4. app folder contains the source code for frontend and backend
5. PartA_TaskA_KnowledgeGraphApp_Instruction.pdf contains brief report explaining the design choices and any challenges faced during 
implementation along with set screenshots that explains the entire flow of the application to be included in the 
report.
6. PartA_TaskB_Enhancement_Plan.pdf contains the enhancement plan.
7. PartB_Literature_Survey_RAG.pdf contains documented literature review for Advancements in Retrieval-Augmented Generation for Knowledge-Enhanced 
Language Models
